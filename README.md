# 🤖 Enigma_Local

Система дистанционного управления роботом на базе **TrackDuino** через защищенный TCP-чат. Проект позволяет передавать команды на контроллер, обмениваясь зашифрованными сообщениями в локальной сети между несколькими пользователями.

---

## 🏗 Архитектура системы

Проект объединяет аппаратную часть и сетевое ПО по модели «Клиент-Сервер»:

* **Firmware (Arduino/C++):** Прошивка для контроллера. Принимает команды через Serial-порт (115200 бод), управляет моторами и встроенным RGB-светодиодом.
* **Server (Python):** Центральный узел. Принимает подключения от клиентов, транслирует зашифрованные сообщения всем участникам чата.
* **Client (Python/Tkinter):** Графический интерфейс. Позволяет переписываться и отправлять команды на локально подключенную плату.

---

## 🌟 Основные возможности

* **Автопоиск сервера:** Клиент автоматически сканирует подсеть `/24` и предлагает список доступных IP-адресов серверов в вашей локальной сети.
* **XOR-Шифрование:** Все сообщения в сети передаются в зашифрованном виде (симметричный ключ `STC_SECRET_KEY`).
* **Приватное управление:** Команды на робота (префикс `TD:`) обрабатываются только тем клиентом, к которому физически подключена плата. Остальные участники видят команду в чате, но не могут управлять чужим устройством.
* **Многопоточность:** Прием данных, сканирование сети и работа GUI разнесены по разным потокам (`threading`).
